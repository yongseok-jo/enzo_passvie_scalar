Enzo user's guide: code requirements

  • About 50 MB of free disk space.
  • The HDF libraries. Assuming you don't want to use HDF4, you should only need the HDF5 library.
  • Fortran 77 or Fortran 90 and C++ compilers.
  • The MPI library.
