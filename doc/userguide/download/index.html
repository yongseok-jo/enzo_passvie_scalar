Enzo download page

Click on this link to obtain the Enzo tarball:

download Enzo

This will download a file named enzo-1.0.1.tar.gz, which contains the source code for Enzo and associated utilities and tools. You can untar this file with the command:

gunzip enzo-1.0.1.tar.gz; tar xvf enzo-1.0.1.tar

The file will untar into a directory called enzo-1.0.1.

At this time please sign up for the Enzo User listserv. Subscribe to the listserv by emailing listserv@ucsd.edu with a blank Subject line, and with a body text of "add (email address) enzo-users-l" (no quotation marks), where (email address) is the email address that you wish to subscribe to. We suggest that all Enzo users subscribe to this list for information on Enzo bug fixes, code updates, and general Enzo-related news. Please send email to this list when you have questions/problems with the code that you feel other users may also encounter.
