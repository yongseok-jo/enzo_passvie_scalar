Contact Information

Enzo User Listserv
The Enzo User listserv is enzo-users-l@ucsd.edu . Subscribe to the listserv by emailing listserv@ucsd.edu with a blank Subject line, and with a body text of "add (email address) enzo-users-l" (without the quotation marks), where (email address) is the email address that you wish to subscribe to. We suggest that all Enzo users subscribe to this list for information on Enzo bug fixes, code updates, and general Enzo-related news. Please send email to this list when you have questions/problems with the code that you feel other users may also encounter.

Enzo Developer Listserv
The Enzo developer listserv is enzo-l@ucsd.edu. If you have any special requests concerning the code, press releases, etc. please email the Developer listserv.

Contact Us!
The director for the Laboratory for Computational Astrophysics is Michael Norman. Please contact the Enzo development group at enzo-l@ucsd.edu if you have any questions or comments.
