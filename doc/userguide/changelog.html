Code Modification History

1 March 2004 - Enzo is released to the public.

28 July 2003 - Enzo is released to friendly users.

1 July 2003 - Enzo bundle is prepared in preparation for release to friendly users. The code bundle includes: enzo, analysis tools (enzo_anyl, enzohop, findpeaks), inits, HDF 5 tools, ring. These packages compile and run on all currently allocated NCSA, SDSC and PSC systems as well as on various sites' Teragrid clusters. Pre-public release change information can be found here.
