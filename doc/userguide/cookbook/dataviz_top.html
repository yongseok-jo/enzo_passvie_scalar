Data Visualization

Several tools are available for making images from Enzo data:

LCA Vision

LCA Vision is a free scientific visualization tool written at the Laboratory for Computational Astrophysics while the group was at The National Center for Supercomputing Applications. It's a very powerful tool custom-designed to visualize AMR data. The LCA Vision home page is here.

Enzo Projection Tool

Enzo can make simple 2D projections (see this page for information on how to make projections). We have a simple script written for IDL (a commercial program made by Research Systems, Inc.) which can be used to convert these images to jpeg.

Jacques

Additionally, one can use a visualization tool called Jacques, which is written by Tom Abel. This code is also based on IDL, and is currently only configured for HDF 4 datasets. However, it can be modified to use HDF 5.

Other tools

We will post other useful visualization tools as they are made known to us.
