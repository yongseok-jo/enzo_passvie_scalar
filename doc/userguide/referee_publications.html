Refereed Publications and Conference Proceedings with Enzo

This is a list of papers in refereed journals and conference proceedings based on research done with the Enzo code. It is not necessarily either complete or up-to-date. If you would like your publication added to this list, please email Brian O'Shea at bwoshea (AT) lanl.gov. It would be greatly appreciated if you would format your submissions in the following html format:

<li>paper title<br>
<em>Authors (last name, first initial)</em><br>
Journal Reference or astro-ph listing</li><br>

Email me a list of your enzo-related publications and I'll gladly put them on this page. If you want to put in links to the paper reference or the astro-ph abstract, that would be fantastic.

Note that papers are listed in reverse chronological order so that the most recent publications should be listed first.
